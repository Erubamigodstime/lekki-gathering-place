## âš¡ Performance Considerations

### Current Implementation
- Direct API calls on component mount
- Refresh on dialog close (success callback)
- Loading states for async operations
- Error boundaries with toast notifications

### Future Optimizations
- **React Query/TanStack Query**: Cache API responses, auto-refetch, optimistic updates
- **Virtual Scrolling**: For large submission lists (100+ items)
- **Pagination**: Load submissions in batches
- **Debouncing**: For search/filter inputs
- **Lazy Loading**: Code split dialog components
- **Image Optimization**: Compress and lazy load certificate images
